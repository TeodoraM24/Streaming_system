# Quick Start Guide

## Prerequisites
1. Backend server running on `http://localhost:8080`
2. Node.js installed

## Start the Frontend

```bash
cd Frontend/StreamflixUI
npm run dev
```

Open `http://localhost:5173` in your browser.

## First Steps

1. **Register a new account**
   - Click "Don't have an account? Register"
   - Fill in all fields
   - Click "Register"

2. **Explore the dashboard**
   - You'll be automatically logged in
   - Use the left sidebar to navigate

3. **Try these features**:
   - **Movies** → View all movies or top-rated
   - **Genres** → Click a genre to filter content
   - **Account** → Edit your personal information
   - **Plans** → View available subscription plans
   - **Subscriptions** → Subscribe to a plan

## Common Actions

### Browse Movies
1. Click "Movies" in sidebar
2. Click "Top Rated" to see best movies
3. Go to "Genres" to filter by category

### Create a Profile
1. Click "Profiles" in sidebar
2. Click "Create Profile"
3. Enter a profile name
4. Click "Create"

### Subscribe to a Plan
1. Click "Plans" to see available options
2. Click "Subscriptions"
3. Select a plan from dropdown
4. Click "Subscribe"

### Write a Review
1. Click "Reviews" in sidebar
2. Click "Write Review"
3. Fill in the form (you'll need Profile ID and Content ID)
4. Click "Submit Review"

## Tips

- **Profile ID**: Go to "Profiles" to see your profile IDs
- **Content ID**: Browse "Movies" or "Shows" to see content IDs
- **Logout**: Click "Logout" in top-right corner
- **Errors**: Check browser console (F12) for detailed error messages
